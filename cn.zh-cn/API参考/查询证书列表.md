# 查询证书列表<a name="ListCertificates"></a>

## 功能介绍

根据证书名称或绑定域名查询证书列表。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SCM&api=ListCertificates)中调试该接口。

## URI
