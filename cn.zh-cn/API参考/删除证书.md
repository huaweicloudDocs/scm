# 删除证书<a name="DeleteCertificate"></a>

## 功能介绍

删除证书实例，即将证书资源从华为云系统中删除。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SCM&api=DeleteCertificate)中调试该接口。

## URI

DELETE /v3/scm/certificates/\{certificate\_id\}
