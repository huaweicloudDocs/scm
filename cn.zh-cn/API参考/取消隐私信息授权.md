# 取消隐私信息授权<a name="ZH-CN_TOPIC_0182547628"></a>

## 功能介绍<a name="section37533920154934"></a>

取消隐私信息授权，删除在SSL证书管理中保存的隐私数据。

## URI<a name="section37627629154934"></a>

-   URI格式

    DELETE /v2/\{project\_id\}/scm/privacy-protection/\{cert\_id\}
