## 示例<a name="section32834116430"></a>

如下以上传名称test的证书为例。

-   请求样例

    ```
    {
        "cert_name":"test",
        "cert":"-----BEGIN CERTIFICATE----- *** -----END CERTIFICATE-----",
        "private_key": "-----BEGIN RSA PRIVATE KEY----- *** -----END RSA PRIVATEKEY-----"
    }
    ```

-   响应样例

    ```
    { 
        "cert_id": " scs1481110651012"
    }
    ```

    或

    ```
    { 
       "error_code": "SCM.XXXX",  
       "error_msg": "XXXX"   
     }
    ```
