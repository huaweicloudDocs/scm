# 获取证书详情<a name="ShowCertificate"></a>

## 功能介绍

查询某张证书的详细信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SCM&api=ShowCertificate)中调试该接口。

## URI
